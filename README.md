# 課題　-- にゃんけん ver.3 

## ①課題内容（どんな作品か）
-  ネコとじゃんけんするアプリ
- 試合記録をブラウザ上で表示したり、ローカルストレージに記録することができる

## ②工夫した点・こだわった点
- ゲームラウンドごとに記録ができるようにした

## ③難しかった点・次回トライしたいこと(又は機能)
- ゲームラウンドごとにローカルストレージにデータ保存するためのキーの設定に非常に苦労した
- 複数のvalueをローカルストレージに保存する方法がわからず苦労した
- ゲーム結果をテーブル表示するさいに、前回記録を消去して新たなデータを追記していく設定が難しかった
- 表に追記していく方法もわかりずらかった
- 表を一行ずつ削除する設定しかできず、ヘッダーと最初の行のみのこして残りを一斉削除する方法を次回は見つけたい
- 今回はとにかくコードをかくうえでのロジックで迷走したうえに、やりたいことがなかなか実装できずに大変だった
